
   StatsCountsMS_Tests - Graphical User Interface with msmsTests functionality

1 - INSTALLATION

    StatsCountsMS_Tests is an R script which may be run directly on R provided
  that you have installed R v 3.3.0 or higher, the package msmsTests and
  its dependencies, and the packages gWidgets and RGtk2.

2 - USER MANUAL

    Please see the user manual provided. 
    Further help may be found in the vignettes and manual of the msmsTests
    package.
